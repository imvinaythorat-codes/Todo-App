# React Todo Application

This is a simple Todo List application built as part of a React assignment using **React** and **Vite**.

## Features
- Add new todos
- Mark todos as completed
- Edit existing todos
- Delete todos
- Clean and user-friendly interface

## Tech Stack
- React
- Vite
- JavaScript
- CSS
